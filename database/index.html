Database-hosting i 2026

Databaser er hjertet i de fleste webapper — alt fra brukerprofiler til produktkataloger til ordrer lagres her. Tradisjonelt måtte du selv installere og vedlikeholde PostgreSQL eller MySQL på en server. I 2026 har managed database-tjenester blitt så gode og billige at 95 prosent av nye prosjekter bruker dem i stedet.

SUPABASE OG NEON DOMINERER 2026

For PostgreSQL (den klart mest populære databasen i 2026) er Supabase og Neon de dominerende valgene. Begge har generøse gratis-tiers som dekker tidlig-stadium-prosjekter. PlanetScale er standarden for MySQL. Redis via Upstash for cache og realtime.

PostgreSQL-leverandører

Supabase — den komplette pakken

Open-source Firebase-alternativ bygget på PostgreSQL. Leverer: database, auth, realtime, storage, edge functions. Gratis-plan: 500 MB database, 50 000 månedlige aktive brukere, 1 GB file storage.

Pro-plan: 25 usd/mnd gir 8 GB database, 100 GB båndbredde, daglige backups i 7 dager. Godt nok for SMB-nettbutikker og apper.

// Supabase-klient
import { createClient } from '@supabase/supabase-js';

const supabase = createClient(
  'https://xyz.supabase.co',
  process.env.SUPABASE_ANON_KEY
);

// Hent data
const { data, error } = await supabase
  .from('produkter')
  .select('*')
  .eq('kategori', 'klaer')
  .limit(20);

Neon — serverless PostgreSQL

Ren PostgreSQL uten ekstra lag. Serverless-arkitektur: skalerer til null når den ikke brukes, starter på under 500 ms. Branching-funksjon lar deg lage kopier av databasen for preview-miljøer.

Gratis: 0,5 GB storage, 191 compute-timer. Pro: fra 19 usd/mnd for 10 GB. Direkte kompatibel med Vercel og Railway.

Vercel Postgres

Vercels managed Postgres, bygget på Neon. Integrert med Vercel-dashboard. Ideell hvis du allerede bruker Vercel.

Railway PostgreSQL

Managed Postgres direkte i Railway. 2 GB RAM koster ~100 usd/mnd — dyrere enn alternativene men svært enkelt hvis du allerede bruker Railway.

DigitalOcean Managed PostgreSQL

Fra 15 usd/mnd for Basic-tier (1 vCPU, 1 GB RAM, 10 GB storage). Multi-AZ tilgjengelig for HA.

Selvhostet på Hetzner

For kostnadsfokuserte og tekniske team: PostgreSQL 15 på Hetzner CX22 (45 kr/mnd). Krever manuelt oppsett av backups, monitoring og HA.

MySQL-alternativer

PlanetScale

Serverless MySQL basert på Vitess (YouTubes database-engine). Skalerer horisontalt til massiv skala. Branching for preview-miljøer. Gratis-plan fjernet i 2024, så nå fra 39 usd/mnd. Fortsatt standard for seriøse MySQL-apper.

Fly.io MySQL

Fly.io tilbyr MySQL via deres ekstensjoner. Fastpris per maskin.

AWS RDS MySQL / Azure Database for MySQL

Enterprise-valg. Dyrt, men med maksimal kontroll og compliance.

Redis og cache-tjenester

Upstash

Serverless Redis og Kafka. Per-request-prising: 0,2 usd per 100K requests. Perfect for session storage, rate limiting, og lightweight cache. Gratis-plan: 10 000 requests/dag.

Railway Redis

Managed Redis i Railway-prosjekter. Fastpris, enkelt oppsett.

AWS ElastiCache / Azure Cache for Redis

Enterprise-Redis. Dyrt men kraftig.

Spesialdatabaser

Turso — SQLite ved edge

libSQL (SQLite-kompatibel) distribuert globalt. Ultra-rask lese-ytelse. Nyttig for read-tunge workloads og edge-applikasjoner. Gratis: 9 GB, 1 milliard rader.

MongoDB Atlas

Dokumentdatabasen som dominerte på 2010-tallet. Fortsatt populær for fleksible data-modeller. Gratis M0-tier med 512 MB.

Clickhouse Cloud

Analytisk database for OLAP-workloads. Ekstremt rask på store datamengder.

Pinecone / Weaviate

Vektor-databaser for AI-applikasjoner (RAG, semantisk søk). Pinecone gratis-tier med 100 000 vektorer.

Hvordan velge database

For SMB-nettbutikk eller webapp

Start med Supabase. Du får database + auth + storage + realtime i én pakke. 25 usd/mnd Pro-plan dekker det meste.

For Next.js-app på Vercel

Vercel Postgres (som er Neon under panseret) gir beste integrasjon. Eller bruk Neon direkte for mer kontroll.

For høy-skala MySQL-app

PlanetScale hvis du allerede har MySQL-stack og forventer stor trafikk.

For AI-app

Kombiner Supabase (brukerdata) med Pinecone (vektor-embeddings).

For kostnadsfokusert prosjekt

Selvhost PostgreSQL + Redis på Hetzner CX22. 45 kr/mnd for alt, krever bare ~2 timers oppsett.

Backup-strategi

Ingen backup er lik data-tap venter. Dette gjelder uansett database:

  • Daglige automatiske backups (Supabase, Neon, Railway gir dette automatisk)
  • Ukentlige snapshots til separat storage (S3, R2)
  • Månedlig restore-test — backup er ikke ekte før du har restoret en
  • Offsite copy — ikke hos samme leverandør

Connection pooling

Serverless-funksjoner kan åpne hundrevis av database-tilkoblinger samtidig, noe som overbelaster Postgres. Løsninger:

  • Supabase: Built-in Supavisor (PgBouncer)
  • Neon: Built-in pooler
  • Vercel + Neon: Edge-pooling automatisk
  • Selvhostet: Installer PgBouncer selv

Prisoversikt for typisk norsk SMB

  • Supabase Pro: 275 kr/mnd — database + auth + storage + realtime
  • Neon Pro + Upstash Redis: 230 kr/mnd
  • PlanetScale Scaler: 430 kr/mnd
  • Selvhostet på Hetzner: 45-90 kr/mnd + tid til vedlikehold
  • AWS RDS medium: 2000+ kr/mnd
